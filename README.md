# What this is
This calculator uses the Euler method in atmosphere rocket equation to estimate final velocity of a craft in atmosphere given certain inputs in the hit video game Kerbal Space Program.
This calculator only works for SRB's at this point as their masses and fuel amounts are fixed.
Thrust is assumed to be 100% on the SRB.
Values are calculated from second to second because I dont know calculus, this causes an inaccuracy of ~10% during testing.
